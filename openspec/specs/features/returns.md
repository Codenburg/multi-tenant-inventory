# Feature: Returns

## Problema
Gestionar devoluciones sin inconsistencias de stock.

## Scope
- seleccionar venta
- seleccionar item vendido
- ingresar cantidad a devolver
- registrar devolución

## Reglas
- no se puede devolver más de lo vendido
- stock se incrementa automáticamente

## Acceptance Criteria
- devolución registrada correctamente
- stock actualizado
- validación de límites funcionando
