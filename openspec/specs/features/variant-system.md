# Feature: Variant System

## Problema
Un producto puede tener múltiples variaciones (talle, color, etc.).

## Scope
- crear variantes
- asignar atributos dinámicos
- gestionar stock por variante
- definir precios por variante

## Reglas
- stock se maneja a nivel variante
- atributos son configurables por tienda
- cada variante tiene precios independientes

## Acceptance Criteria
- variantes correctamente asociadas a producto
- atributos visibles
- stock independiente por variante
