# Feature: Metrics Dashboard

## Problema
El usuario no tiene visibilidad del rendimiento del negocio.

## Scope
- ventas mensuales
- ingresos totales
- productos más vendidos

## Reglas
- métricas deben basarse en datos reales de ventas
- no usar valores simulados

## Acceptance Criteria
- métricas correctas
- consistentes con datos de ventas
- carga rápida
